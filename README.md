右クリック一発でテキストファイル化して保存するchrome拡張機能です

## 📥 ダウンロード
ここからZipファイルをダウンロードできます！
[ai-text-saver_v1.0.zip](https://github.com/kazudon151/text-copy/releases/download/v1.0/text-copy.zip)

## 使い方
1. ブラウザ上で、保存したいテキストを選択する
2. 右クリックして「選択範囲を.txtとして保存」を押す
3. 即座にダウンロードされます！

## 導入手順
Chrome公式ストアには置いていないので、以下の手順で入れてください。

1. ダウンロードしたZipを解凍する
2. ChromeのURL欄に `chrome://extensions/` と入力して開く
3. 右上の「デベロッパーモード」をONにする
4. 左上の「パッケージ化されていない拡張機能を読み込む」から、解凍したフォルダを選ぶ
